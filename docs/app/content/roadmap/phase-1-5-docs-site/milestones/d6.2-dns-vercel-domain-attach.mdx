## Why This Milestone Exists

DNS `docs` CNAME is configured (grey cloud) but Vercel project has no production domain — visitors get 404 at docs.ibexharness.com.

Production attach is intentionally last: only after preview review (D.4.1), CI gates (D.4.2), and visual QA (D.5.1).

Apex and www records follow ibexdepo landing pattern; docs subdomain points to Vercel.

- Orange-cloud proxy on docs CNAME (DNS-only per brief)
- Wildcard subdomains beyond docs/www

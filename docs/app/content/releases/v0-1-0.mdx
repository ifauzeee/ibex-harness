First public release of the IBEX Harness proxy service.

## Highlights

- Bearer token validation via gRPC to the auth service
- Agent verification against organization scope
- Redis-backed rate limiting
- JSON body normalization ahead of provider adapters

## Upgrade notes

Use `make db-migrate` and `make db-seed` after pulling this release for a local stack with seeded test credentials.

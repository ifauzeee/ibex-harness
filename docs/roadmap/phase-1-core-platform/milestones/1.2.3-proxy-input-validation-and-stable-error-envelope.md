# Milestone 1.2.3: Proxy Input Validation and Stable Error Envelope

**Status:** Planned
**Goal:** [1.2 Proxy platform integration](../goals.md)
**Phase:** 1 — Core Platform

## Why This Milestone Exists

Milestone 1.2.2 mentions "Validate Content-Type, body size limits per SECURITY.md"
as a single task item. But `SECURITY.md §5.2` and `AGENTS.md §5.2` define a
comprehensive input validation requirement:

- string length bounds
- array size bounds
- JSON object size bounds
- enum strict validation (no implicit coercion)
- UUID format validation
- numeric bounds (confidence 0..1, etc.)

`API_DOCUMENTATION.md` defines a stable error envelope that every endpoint must
return. Without a dedicated milestone, this becomes an afterthought bolted onto
1.2.2, and the error envelope will be inconsistent across endpoints.

This is a security requirement: unbounded inputs are a DoS vector. An oversized
JSON body can exhaust memory. An unbounded array can cause O(n²) processing.

### 3. Implement stable error envelope

Per `API_DOCUMENTATION.md`:

```go
// ErrorResponse is the stable error envelope for all API errors.
// Every error response from every endpoint must use this structure.
type ErrorResponse struct {
    Error ErrorDetail `json:"error"`
}

type ErrorDetail struct {
    Code      string       `json:"code"`
    Message   string       `json:"message"`
    Detail    string       `json:"detail,omitempty"`
    DocsURL   string       `json:"docs_url,omitempty"`
    RequestID string       `json:"request_id"`
    Timestamp time.Time    `json:"timestamp"`
    Fields    []FieldError `json:"field_errors,omitempty"`
}

type FieldError struct {
    Field   string `json:"field"`
    Code    string `json:"code"`
    Message string `json:"message"`
}

// Error codes (stable — never change these strings)
const (
    ErrCodeValidation       = "VALIDATION_ERROR"
    ErrCodeUnauthorized     = "UNAUTHORIZED"
    ErrCodeForbidden        = "FORBIDDEN"
    ErrCodeNotFound         = "NOT_FOUND"
    ErrCodeConflict         = "CONFLICT"
    ErrCodeRateLimited      = "RATE_LIMITED"
    ErrCodeServiceUnavailable = "SERVICE_UNAVAILABLE"
    ErrCodeInternalError    = "INTERNAL_ERROR"
    ErrCodeProviderError    = "PROVIDER_ERROR"
)

// WriteError writes a stable error response to the ResponseWriter.
// It sets Content-Type, status code, and JSON body.
// It never leaks internal error details to the client.
func WriteError(w http.ResponseWriter, r *http.Request, status int,
    code, message string, fields ...FieldError)
```

### 4. Implement request validator for chat completion

```go
// ValidateChatCompletionRequest validates a parsed chat completion request.
// Returns a slice of FieldErrors (empty if valid).
func ValidateChatCompletionRequest(req *ChatCompletionRequest) []FieldError {
    var errs []FieldError

    if req.Model == "" {
        errs = append(errs, FieldError{
            Field: "model", Code: "REQUIRED",
            Message: "model is required",
        })
    } else if len(req.Model) > MaxModelNameLength {
        errs = append(errs, FieldError{
            Field: "model", Code: "TOO_LONG",
            Message: fmt.Sprintf("model must be at most %d characters", MaxModelNameLength),
        })
    }

    if len(req.Messages) == 0 {
        errs = append(errs, FieldError{
            Field: "messages", Code: "REQUIRED",
            Message: "messages array must not be empty",
        })
    } else if len(req.Messages) > MaxMessagesPerRequest {
        errs = append(errs, FieldError{
            Field: "messages", Code: "TOO_MANY",
            Message: fmt.Sprintf("messages array must have at most %d items", MaxMessagesPerRequest),
        })
    }

    for i, msg := range req.Messages {
        if msg.Role == "" {
            errs = append(errs, FieldError{
                Field: fmt.Sprintf("messages[%d].role", i),
                Code: "REQUIRED", Message: "role is required",
            })
        }
        if !isValidRole(msg.Role) {
            errs = append(errs, FieldError{
                Field: fmt.Sprintf("messages[%d].role", i),
                Code: "INVALID_ENUM",
                Message: "role must be one of: system, user, assistant, tool",
            })
        }
        if len(msg.Content) > MaxMessageContentBytes {
            errs = append(errs, FieldError{
                Field: fmt.Sprintf("messages[%d].content", i),
                Code: "TOO_LONG",
                Message: fmt.Sprintf("content must be at most %d bytes", MaxMessageContentBytes),
            })
        }
    }

    return errs
}
```

## What Ships Today

<Callout type="note" title="Current capability boundary">
  The platform layer is production-ready for authenticated request handling and
  governance. Provider forwarding and memory pipeline features remain out of
  scope until Phase 2/3.
</Callout>

### Platform (Phase 1)

- Auth gRPC (`ValidateToken`, `ValidateAgent`) with Argon2id-backed token validation
- Proxy auth middleware, agent verification, request normalization, stable error envelopes, rate limiting
- Security integration gate with 31-case cross-tenant denial matrix and CI enforcement
- Shared packages (`logger`, `metrics`, `telemetry`, `config`, `apierror`, `shutdown`, `reqid`, `ratelimit`, `crypto`, `permissions`)
- Postgres migrations with RLS; integration smokes for auth, proxy, and security paths
